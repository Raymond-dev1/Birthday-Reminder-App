## Birthday-Reminder-App 

This app sends mail greetings to a celebrating user.

# Technologies 

- Nodejs
- MongoDB
- Nodemailer 
- Express 

# Description 

it runs on a cron job and loops through database every morning for the celebrating user(s),
and sends mail greetings with nodemailer

This project fosters  my understanding of CRON JOBS in real life use cases. 

# Installation 

- Node latest version installed 
- npm install 
- npm run dev
